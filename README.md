# Placar Multi-Esportes (Padrão FIAS) ⏱️🥋

Um sistema de placar eletrônico de código aberto, de alta precisão e responsivo, desenvolvido para atender competições de artes marciais, com suporte inicial ao padrão oficial da Federação Internacional de Sambo (FIAS).

## 🎯 Objetivo

Fornecer à comunidade esportiva, incluindo a Confederação Brasileira de Sambo (CBSA), uma ferramenta gratuita, resiliente e adaptável a qualquer tela (de smartwatches a telões), garantindo o gerenciamento perfeito de tempo e pontuação.

## 🛠️ Tecnologias Utilizadas

* **Python 3.10+**
* **Flet** (Interface gráfica reativa baseada em Flutter)
* **uv** (Gerenciamento ultrarrápido de dependências e ambientes virtuais)

## 🚀 Como Rodar Localmente

1.Clone o repositório:

```bash
   git clone [https://github.com/atnzpe/placar-eletronico.git](https://github.com/atnzpe/placar-eletronico.git)

   
cd placar-cbsa
```

2.Sincronize o projeto usando o uv:

```bash
uv sync
```

3.Execute o aplicativo:

```bash
uv run flet run main.py
```
